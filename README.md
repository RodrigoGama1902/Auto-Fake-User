# Auto-Fake-User

Blender Add-on that will automatically enable Fake User for user selected data blocks, such as materials, objects and more.

# How to install

Simply install the add-on .zip file

# Where it will be

View3D > Sidebar > Tool Tab > Auto Fake User

# How to use

1 - Enable Auto Fake User
2 - Select all data blocks you want to be Fake User
3 - Done, every time you save this file, these data blocks will be set to Fake User
